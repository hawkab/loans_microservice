## Loan claims micro-service (REST API CRUD with Spring Boot and H2).

### Technology stack:

* Spring Boot;
* Spring Web;
* Spring Data;
* H2 database;
* Hibernate;
* Spring Security (as basic authentication).

#####To run this application use:

```bash
mvn spring-boot:run
```
